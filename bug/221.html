Comment 1 Mark Longair 2010-08-18 12:20:43 CDT 
One way of reproducing this is to take the following steps:

 - Create a completed path ("Path (0)")

 - Select "Path (0)"

 - Start a path at another point in the stack and click (with the join modifier key) on "Path (0)".

 - Click "No" to cancel this temporary path

 - Click "Cancel" to cancel the second path entirely

 - Click on "Export to SWC", and the error is produced.
