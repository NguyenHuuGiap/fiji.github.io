Description Bryan Crawford 2010-09-16 19:46:18 CDT 
Hello Fijians.

I've been very happily using Fiji for quite a while, and ImageJ for many years before that.  But during the past few weeks (I'm afraid I didn't notice at what version this started happening), I've been encountering either some bugs with lookup tables, or a change in how these are defined that I don't know how to deal with.

Firstly, if I open a 16-bit confocal stack (Leica SP2... opened via the .lei file), it inevitably displays all focal planes as black, with an indication in the title bar of the window that it is using an inverted LUT.  I can go to Image->Lookup Tables and select greys and everything is fine, but I can't figure out how to stop it from doing this... it never used to.

Secondly, and even more problematically, in the current version of Fiji, if I open a stack and run Image->Stacks->3D project the resulting rotation is completely munged.  Again, it switches to and inverted lut, but because it's putting the projections on a black background I can't just switch the LUT to fix it.  Fortunately, I have an older version of Fiji on another computer (which uses ImageJ 1.44a) and these problems don't manifest on it.

Let me know if (and how) you want me to send example stacks.

Thanks for your help!
