Comment 1 Wayne Rasband 2015-11-30 15:52:20 CST 
The File>Import>Image Sequence command uses the ImageJ directory chooser, so you have to select the directory that contains the PNG images. You can also drag and drop the folder containing the images onto the "ImageJ" window.
