The "Stack Histogram" function in "Enhance Contrast" does not work properly. It seems to take the single slice histogram rather than the complete stack. Note that all tickmarks are on, except for the "equalization". In ImageJ's current version, the same thing happens. In an older version, 1.43u, this problem does not yet exist.

The origin of the problem might lie in a bugfix (as of version 1.46g) for Enhance Contrast WITH histogram equalization, which has been implemented as noted here: http://imagej.1557.n6.nabble.com/Equalize-Histogram-on-whole-volume-not-working-td4373815.html

Yours sincerely,

Maarten Voorn
