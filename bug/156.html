Comment 1 Johannes Schindelin 2011-03-23 15:15:05 CDT 
This should be fixed now, in multiple ways: the original plugin is no longer compiled for 1.6, we default to 1.6 compatibility mode, and if you still run into problems, the class loader should offer to convert the classes to 1.5 compatibility (you still need to restart Fiji after that...).
