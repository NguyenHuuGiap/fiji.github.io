Comment 3 Curtis Rueden 2015-06-03 06:08:22 CDT 
Gabriel, you probably turned off SCIFIO on the non-working installation.

Edit > Options > ImageJ2
Use SCIFIO when opening files (BETA!)

With that option on, I can open your file. With it off, it tries to use ImageJ1's I/O logic and reports the "Not a BMP file" error you mentioned.

Which version of BF you are using doesn't affect this.
