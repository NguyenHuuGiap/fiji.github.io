Comment 1 Curtis Rueden 2016-04-27 09:04:51 CDT 
Thanks for the report, Vytas.

ClearVolume requires the Java-8 update site to be enabled.

From the ClearVolume wiki page:

"If you use an older Fiji installation you MUST also check and install the Java-8 update site."

Alternately, you can download a fresh Fiji and it will come with Java-8 already turned on.

If you turn on the Java-8 update site and it still doesn't work, definitely let us know.
