Description olivia.viessmann 2014-04-18 06:36:42 CDT 
I used the SNT to segment vessels in MR data. I created a tree structure and exported it as SWC. I loaded the swc file in the trees toolbox (Matlab based) which can calculate for example the angle between the branches. Comparing the angle values with my tree, some angles didn't make sense. By the eye I could tell, that the angle was smaller then 90 degrees, but the trees toolbox gave me , for example 120. In other cases it seemed to match. I then loaded the SWC file in the neuromantic software and took a close look at the branches and saw that for some branches the branch didn't start at the mother path, but crossed it (creating a secon angle, which is 180-true angle). Talking to a friend, he told me he already experienced the same behaviour in the SNT, that sometimes branches do not start correctly at the mother path. I could send you the SWC file if you like. 
Thank you very much and lot's of sunshine,
Olivia
