Description Jean Livet 2012-06-12 13:51:23 CDT 
Following Fiji updates in the last weeks, i noticed a minor bug occuring on two different systems (Windows 32bit and 64bit) with color images:
After making a 3 channel composite using the 'merge channel' tool, Fiji  improperly saves this image as a 48-bit color tif .
48-bit tif images generated in this way can be opened as color images with Fiji itself, but not other with other programs such as Photoshop. Instead Photoshop only sees the 1st channel and opens it as a 16-bit black and white image.

Comment 1 Wayne Rasband 2012-06-12 17:07:23 CDT 
3 channel 16-bit composite color images are saved as Photoshop-compatible 48-bit RGB TIFFs if the slice label of the first channel is "Red". You can set the slice label and save the image in a macro using

   setSlice(1);
   setMetadata("label", "Red");
   saveAs("tiff", "");

-wayne
