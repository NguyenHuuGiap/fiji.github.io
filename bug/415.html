Description Curtis Rueden 2012-03-16 20:56:07 CDT 
When writing code in the Script Editor, pressing shift+space invokes autocompletion. I would like the option to disable this behavior, because it often results in the loss of spaces when typing quickly. In particular, at least on U.S. keyboards, to type curly braces ('{' and '}') you must hold shift. So I often try to type:

   public void run(String arg) {

but it comes out like:

   public void run(String arg){

because I press shift slightly before pressing the space before the curly brace.
